---

## Tools

### File Organizer (file_organizer.py)
Automatically sorts files into categorized 
folders based on file extension.
Skills: File I/O, os module, automation logic

### Password Generator (password_generator.py)
Generates strong randomized passwords with 
customizable length and character sets.
Skills: Random module, string manipulation, 
security best practices

### Network Scanner (network_scanner.py)
Scans a local network to discover active hosts 
and open ports. Used in penetration testing 
and network auditing.
Skills: Socket programming, networking, 
cybersecurity recon
